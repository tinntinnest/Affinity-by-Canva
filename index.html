Rastergraafika harjutused - Affinity by Canva

Sissejuhatus

Need harjutused on koostatud Affinity by Canva kasutamiseks. Affinity ühendab endas kolm võimsat rakendust (Pixel, Vector ja Publisher) ühes keskkonnas. Nende harjutuste fookus on rastergraafika töötlemisel (Pixel persona), mis sobib fotode redigeerimiseks, digitaalse kunsti loomiseks ja õppematerjalide ettevalmistamiseks.

Kellele mõeldud?

  • Õpetajatele, kes soovivad luua visuaalseid õppematerjale
  • Õpilastele, kes õpivad digitaalset fototöötlust
  • Algajatele, kes tutvuvad rastergraafika põhitõdedega

Harjutus 1: Töökoha seadistamine ja Persona valik

Eesmärk

Tutvuda Affinity by Canva liidesega ja õppida kasutama Pixel Persona't.

Sammud:

  1. Ava Affinity by Canva
    • Käivita rakendus oma arvutis
    • Märkad üleval vasakus nurgas Persona valikuid: Pixel, Vector, Export
  2. Vali Pixel Persona
    • Kliki üleval vasakul Pixel ikoonil (foto sümbol)
    • See aktiveerib rastergraafika töötlemise režiimi
    • Märkus: Vector Persona on vektorgraafika jaoks, meie kasutame Pixel'it
  3. Loo uus dokument
    • Vali File > New või vajuta Ctrl+N (Windows) / Cmd+N (Mac)
    • Avaneb New Document dialoog
    • Määra parameetrid:
      • Type: Vali "Web" või "Photo"
      • Width: 1920 px
      • Height: 1080 px
      • DPI: 72 (ekraanile) või 300 (trükiks)
      • Color Format: RGB/8
      • Create artboard: Eemalda märge (praegu pole vaja)
      • Background: White (valge) või Transparent (läbipaistev)
  4. Tutvuge tööruumiga
    • Toolbar (Tööriistariba) - vasakul küljel (vertikaalriba)
    • Studio Panels (Paneelid) - paremal küljel
    • Canvas (Lõuend) - keskel (sinu tööala)
    • Context Toolbar (Kontekstuaalne riba) - üleval, muutub valitud tööriista järgi
    • Persona Switcher (Persona vahetaja) - üleval vasakul
  5. Olulised paneelid Pixel Persona's
    • Kui paneelid ei ole nähtavad, ava View > Studio:
    • Layers - kõik kihid ja nende järjestus
    • History - tehtud toimingute ajalugu (tagasivõtmiseks)
    • Color - värvide valimine ja kohandamine
    • Swatches - salvestatud värvipalett
    • Navigator - kiire navigeerimine suurtel piltidel
    • Histogram - pildi heledusjaotus
    • Brushes - pintsli valikud
  6. Tööruum lihtsustamine
    • Vajuta Tab klahvi - peidab/näitab paneelid
    • Lohista paneele, et neid ümber paigutada
    • Parem klõps paneelile > Close sulgemiseks

Harjutus 2: Valiku tööriistad (Selection Tools)

Eesmärk

Õppida kasutama erinevaid valiku tööriistu.

Tööriistad:

  • Rectangular Marquee Tool (Ristkülikukujuline valik) - M
    • Vali tööriist toolbar'ist
    • Lohista lõuendile ristkülik
    • Hoia Shift all, et luua ruut
    • Hoia Alt all, et tõmmata keskpunktist
  • Elliptical Marquee Tool (Elliptiline valik) - M
    • Vajuta M uuesti, et vahetada tööriistu
    • Lohista ellips
    • Hoia Shift all ringi jaoks
  • Freehand Selection Tool (Vabakäeline valik) - L
    • Joonista vabakäeline valik
    • Topeltkliki või vajuta Enter, et sulgeda valik
  • Selection Brush Tool (Valikupintsel)
    • Värvi üle ala, mida soovid valida
    • Muuda pintsli suurust [ ja ] klahvidega

Valiku modifitseerimine:

  • Lisa valikule: Hoia Shift all
  • Lahuta valikust: Hoia Alt all
  • Lõika valik: Hoia Shift+Alt all
  • Tühista valik: Ctrl+D / Cmd+D
  • Inverteeri valik: Ctrl+Shift+I / Cmd+Shift+I

Harjutus 3: Kihtide kasutamine (Layers)

Eesmärk

Mõista kihtide loogikat ja kasutamist.

Sammud:

  1. Loo uus kiht
    • Ava Layers paneel (paremal)
    • Kliki + nuppu all
    • Või kasuta Ctrl+Shift+N / Cmd+Shift+N
  2. Nimeta kiht ümber
    • Topeltkliki kihi nimel
    • Sisesta uus nimi
  3. Kihi järjekorra muutmine
    • Lohista kihte Layers paneelil üles-alla
  4. Kihi nähtavus
    • Kliki kihi kõrval olevat silma ikooni
  5. Kihi lukustamine
    • Kliki luku ikoonil
  6. Kihi läbipaistvus (Opacity)
    • Muuda Opacity liugurit Layers paneelil
    • Väärtused 0% (täiesti läbipaistev) kuni 100% (täiesti läbipaistmatu)
  7. Blend Mode (Segunemisrežiim)
    • Ava dropdown Layers paneelil
    • Proovi erinevaid režiime:
      • Normal - tavaline
      • Multiply - korrutamine
      • Screen - ekraan
      • Overlay - ülekate

Praktiline ülesanne õpetajatele:

Projekti nimi: Lihtne infograafika taust

  1. Loo uus dokument (1200 x 800 px)
  2. Loo 3 kihti ja värvi need pintsliga erinevatesse värvidesse:
    • Kiht 1: Sinine taust
    • Kiht 2: Roheline keskmine ala (Blend Mode: Overlay, Opacity: 70%)
    • Kiht 3: Kollane ring (Blend Mode: Screen, Opacity: 50%)
  3. Salvesta projekt kui .afphoto fail
  4. Mõttekoht: Kuidas saaksid seda kasutada tunni slaidide taustal?

Harjutus 4: Joonistamine ja Painting Tools

Paintbrush Tool (Pintsel) - B

Põhiseaded:

  • Width: Pintsli suurus
  • Hardness: Kõvadus (0% = pehme äär, 100% = terav äär)
  • Opacity: Läbipaistvus
  • Flow: Voog (värvi intensiivsus)

Kiirklahvid:

  • [ - Vähenda pintsli suurust
  • ] - Suurenda pintsli suurust
  • Shift+[ - Vähenda kõvadust
  • Shift+] - Suurenda kõvadust

Eraser Tool (Kustukumm) - E

Kustutab pixels kihi piires. Sama seaded nagu pintslil.

Paint Bucket Tool (Värviämber) - G

Täidab valitud ala ühtlase värviga.

  • Tolerance: Määrab, kui sarnaseid värve täidetakse

Gradient Tool (Gradienditööriist) - G

Gradienditüübid:

  • Linear
  • Radial
  • Elliptical
  • Conical

Kasutamine:

  1. Vali Gradient Tool
  2. Vali värvid Swatches või Color paneelist
  3. Lohista lõuendil, et luua gradient

Harjutus 5: Värvide valimine

Color Panel

  1. Ava Color paneel
    • View > Studio > Color
  2. Värvi valimine:
    • Kliki värviringil
    • Või sisesta täpsed väärtused:
      • RGB (Red, Green, Blue)
      • HSL (Hue, Saturation, Lightness)
      • Hex kood (#RRGGBB)

Eyedropper Tool (Pipett) - I

Kliki mis tahes värvile lõuendil, et see valida. Toimib ka pildil.

Harjutus 6: Teksti lisamine

Text Tool - T

  1. Kliki lõuendile
    • Või lohista tekstiboksi
  2. Character paneel:
    • Vali font
    • Määra suurus
    • Määra värv
    • Määra ridade vahe (leading)
    • Määra tähtede vahe (tracking)
  3. Paragraph paneel:
    • Määra joondus (vasak, keskele, parem)
    • Määra taanded
  4. Tekstiefektid:
    • Layer > Layer Effects
    • Lisa vari (Drop Shadow)
    • Lisa välisjooned (Outline)
    • Lisa sisemine vari (Inner Shadow)
    • Lisa helendus (Glow)

Harjutus 7: Filtrid ja Efektid

Filters menüü

  • Blur (Hägusus)
    • Filters > Blur > Gaussian Blur
    • Muuda Radius väärtust
  • Sharpen (Teravustamine)
    • Filters > Sharpen > Unsharp Mask
  • Noise (Müra)
    • Filters > Noise > Add Noise
  • Distort (Moonutamine)
    • Filters > Distort > Wave
    • Filters > Distort > Twirl

Live Filters

Lisa Layer > New Live Filter Layer. Efekt on mittedestruktiivne ja redigeeritav.

Harjutus 8: Foto retušeerimine

Clone Brush Tool (Kloonimispintsel) - S

  1. Vali töökohaga
  2. Hoia Alt all ja kliki allikaalale
  3. Joonista sihtalale

Healing Brush Tool (Paranduspintsel) - J

Sarnane kloonimispintslile. Sobitab tekstuuri ja värvitoone.

Spot Removal Tool

Ideaalne väikeste defektide eemaldamiseks. Kliki defektile ja lohista.

Blemish Removal Tool

Tööriist näonahale. Eemaldab kiiresti väiksed vead.

Harjutus 9: Maskid

Layer Masks

  1. Loo mask
    • Vali kiht
    • Kliki Mask Layer nuppu Layers paneelil
    • Või Layer > Mask Layer
  2. Maski redigeerimine
    • Vali mask Layers paneelil
    • Kasuta pinstlit:
      • Must peidab
      • Valge näitab
      • Hall poolläbipaistev
  3. Maski kustutamine
    • Parem klõps maskil > Delete

Clipping Masks

  1. Vali ülemine kiht
  2. Layer > Insert Inside Selection või Ctrl+Alt+G

Harjutus 10: Shapes ja Vector Tools

Rectangle Tool - M

  1. Vali töökohaga
  2. Lohista lõuendile
  3. Hoia Shift all ruudu loomiseks

Ellipse Tool - M

Lohista ellips. Hoia Shift all ringi jaoks.

Pen Tool - P

Loo kohandatud vektorikujundeid. Kliki punktide lisamiseks. Lohista käepidemeid kõverate jaoks.

Node Tool - A

Redigeeri vektorkujundite punkte (node). Muuda kuju struktuuri.

Harjutus 11: Foto kohandamine (Adjustments)

Adjustments menüü

  • Brightness/Contrast
    • Layer > New Adjustment > Brightness/Contrast
  • Hue/Saturation
    • Muuda värvitoone ja küllastust
  • Levels
    • Täpsem heleduse ja kontrastsuse juhtimine
    • Must-, hall- ja valgepunkt
  • Curves
    • Professionaalne toonide kohandamine
    • RGB kanalid eraldi
  • Black & White
    • Teisenda foto mustvalges
    • Kontrolli värvi kanalite mõju
  • Vibrance
    • Suurenda värvide intensiivsust

Harjutus 12: Eksportimine

Ekspordi pilt

  1. Vali File > Export või Ctrl+Alt+Shift+S
  2. Vali formaat:
    • JPEG: Fotodele, väiksem failisuurus
      • Quality: 80-100%
    • PNG: Läbipaistvusega pildidele
      • Vali PNG-8 või PNG-24
    • TIFF: Professionaalseks kasutamiseks
    • GIF: Animatsioonideks
    • PSD: Photoshopile
  3. Määra suurus ja resolutsioon
    • Width/Height: Pikslites
    • DPI: 72 (web), 300 (print)
  4. Export settings:
    • Area: Whole document või Selection
    • Resample: Bicubic (parimat kvaliteeti)

Harjutus 13: Lõikamise ja muutmise tööriistad

Crop Tool - C

  1. Vali Crop Tool
  2. Lohista käepidemeid
  3. Vajuta Enter kinnitamiseks

Seaded:

  • Unconstrained: Vaba suhe
  • Original Ratio: Säilita algne suhe
  • 1:1, 4:3, 16:9: Standardsed suhted

Move Tool - V

Liiguta kihte ja objekte. Hoia Shift all, et liikuda otsejoones.

Transform Tools

Free Transform - Ctrl+T / Cmd+T

  • Muuda suurust
  • Pööra
  • Moonuta

Hoia:

  • Shift - Säilita proportsioonid
  • Alt - Keskpunktist
  • Ctrl - Moonutamiseks (distort)

Harjutus 14: Projekti salvestamine

Salvesta töötav fail

  1. File > Save - Ctrl+S / Cmd+S
    • Salvesta .afphoto formaadis
    • Säilitab kõik kihid ja seaded
  2. File > Save As
    • Salvesta uue nimega

Automaatne salvestamine

Edit > Preferences > General

  • Märgi Autosave every X minutes
